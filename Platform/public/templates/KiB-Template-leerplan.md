## 3. Retrospective Format

### Cyclus Retrospective Agenda

| Tijd | Onderdeel | Methode |
|------|-----------|---------|
| 0:00 | Check-in | Ronde: Hoe kom je binnen? |
| 0:15 | Terugblik feiten | Timeline gezamenlijk |
| 0:30 | Wat ging goed? | Post-its, clusteren |
| 0:50 | Wat kan beter? | Post-its, clusteren |
| 1:10 | Pauze | |
| 1:20 | Root cause analyse | 5x Waarom |
| 1:40 | Acties bepalen | SMART formuleren |
| 1:55 | Check-out | Ronde: Wat neem je mee? |

### Vragen per Thema

**Thema: Kiezen**
- Hebben we de juiste keuzes gemaakt?
- Missen we iets in scope?

**Thema: Vormgeven**
- Klopt onze visie nog?
- Zijn baten nog relevant?

**Thema: Organiseren**
- Werkt de governance?
- Zijn rollen duidelijk?

**Thema: Sturen**
- Sturen we op de juiste dingen?
- Zijn stuurparameters effectief?

**Thema: Beslissen**
- Worden besluiten tijdig genomen?
- Is escalatie effectief?

**Thema: Samenwerken**
- Werkt stakeholder engagement?
- Bereikt communicatie de doelgroep?

**Thema: Leiden**
- Is er voldoende eigenaarschap?
- Tonen we het juiste leiderschap?

**Thema: Ontwikkelen**
- Leren we voldoende?
- Borgen we kennis?
